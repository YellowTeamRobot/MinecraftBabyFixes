# MinecraftBabyFixes
## Bedrock Edition
Simply add the .mcpack resource pack and enable like normal. Includes both texture and model tweaks.
## Java Edition
#### Coming Soon
In vanilla minecraft, this pack will only perform the *texture tweaks* changes. To enable the model tweaks, optifine or an optifine CEM (custom entity model) compatibility mod is required.

---
### Texture Tweaks
**Added sclera to baby:**
- cat/ocelot
- wolf
- cows
- pigs
- foxes
- sheep
### Model Tweaks
- Baby Horse:
  - face "tapers" instead of being a brick
  - longer legs, because baby horses are lanky
- Baby Mule/Donkey:
  - face "tapers" instead of being a brick
- Baby villager/zombie villager
  - nose slightly longer, instead of being cube shaped
